## 👨🏽‍💻 Stack

- [x] ReactJS ( utilizando o Next.JS para criação do projeto )
- [x] Typescript
- [x] TailwindCSS
- [x] Lucide React (Lib de icons)
## Executando o projeto

Faça o clone deste projeto em seu computador e depois execute com npm:

```bash
 git clone {url}
 cd spotify-tailwindcss
 npm install
 npm run dev
```
